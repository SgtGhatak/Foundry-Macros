# Foundry-Macros
A collection of my macros created for the use in Foundry

Required modules:
- Foundry Core
- JB2A Patreon Complete Collection
- Jack Kerouac's Animated Spell Effects: Cartoon
- Sequencer
- Warp Gate
- Tagger
- Advanced Macros
- FXMaster
