# Foundry-Macros
A collection of my macros created for the use in Foundry V10

Required modules:
- Foundry Core V10
- JB2A Patreon Complete Collection
- Jack Kerouac's Animated Spell Effects: Cartoon
- Sequencer
- Warp Gate
- Tagger
- Advanced Macros
- FXMaster
